XArrow is the wrapper(father) component of the components that would compose the arrow body,head, labels ,etc,
and it provides a shared context for all the components that would be used inside the arrow.

```jsx
<XArrow ... >
  <XLine ... />
  <XHead ... />
   ...
</XArrow>
```
You will use the component XArrow in order to build your own arrow components, with your preferred style and functionality.
